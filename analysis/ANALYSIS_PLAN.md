## 0. 입력 정의 (사전 고정)

- **LLM 점수** = 피평가자 `misstatement_probability` (0–100).
- **1차 비교 프레임 = 원본 vs 원본**: 실험군 8 원본 점수(runs/main, 동결) vs
  대조군 22 원본 점수(runs/rp09/scores — 발사 후 생성). 근거: §5-2 동일
  프로토콜 (양군 모두 정체 노출 원본 페이로드).
- **2차(감도) 프레임 = 교란 실험군 vs 원본 대조군** (RP-05/RP-06의 J14 비대칭
  프레임 — 정체 암기 기여 차단 방향). 두 프레임 결과를 병기하며, 두 프레임의
  결론 규칙 판정이 갈리면 **보수적인 쪽(더 약한 주장)이 헤드라인**이다.
- INCOMPLETE 케이스(재시도 후 실패): 1차 분석에서 제외하고 N을 명기, 제외
  민감도(최악 대체: 대조군은 100, 실험군은 0으로 대체) 병기.

## 1. 1차 통계 검정 (사전 고정)

- **순열 검정, 평균 점수 차** (fraud 8 vs control 22): 100,000 순열,
  seed = 20260707, 단측 (fraud > control). 유의 임계 α = 0.05.
- **2차: Fisher exact** (단측), 2×2 = 그룹 × 플래그. **플래그 임계 = p ≥ 50**
  — 사전 동결 근거: 채점 루브릭(freeze 82a7717 계열)이 실험군 p≥50을 검출
  크레딧(밴드 1+), 대조군 p≥50을 오탐(밴드 0, "false positive on clean
  company")으로 정의한다. 새 임계 발명이 아니라 동결 루브릭의 임계 재사용.

## 2. 효과 크기·시각화 (사전 고정)

- **Cliff's delta** + rank-biserial (동치 관계 명기), **AUC** + 부트스트랩
  95% CI (10,000 재표집, seed 20260707) — N=30 불안정 라벨 의무.
- 1차 시각화 = **30사 점수 점/순위 플롯 전체 공개** (요약 통계가 아니라
  전 데이터 포인트가 독자의 검증 경로).

## 3. FPR 보고 규칙 (사전 고정)

- **"0% FPR" 표기 금지.** FP=0/22이면 단측 95% 상한 = 3/22 ≈ **13.6%**
  (rule of three)로 보고. FP≥1이면 Clopper-Pearson 정확 구간.

## 4. 결론 규칙 (R1–R4 — 헤드라인은 이 규칙이 결정한다)

- **R1 (귀무)**: 1차 순열 p ≥ 0.05 → 헤드라인 = **귀무 결과**, 능력 주장
  없이 그대로 발행.
- **R2 (기계 신호 복제)**: 어느 기준선 b ∈ {Beneish M, Dechow F}에 대해
  Spearman ρ(LLM, b) ≥ **0.7** **이고** 잔차 증분 검정 p ≥ 0.05 (잔차 = 30사
  전체에서 LLM 점수를 b에 OLS 회귀한 잔차; 그 잔차의 그룹 평균차 순열 검정,
  100,000회, seed 동일) → 헤드라인 = "LLM이 알려진 기계 신호를 복제한다",
  "사기를 탐지한다" 아님.
- **R3 (암기 지배)**: 실험군 8 중 **5건 이상**에서 |Δmean(원본−교란, RP-07
  완결 k=5 평균)| ≥ 0.5 × (해당 케이스 원본 점수 − 대조군 원본 중위값) →
  헤드라인 = "점수의 상당 부분은 분석이 아니라 암기를 반영".
  (분모 ≤ 0인 케이스는 자동으로 R3 카운트에 산입 — 분리 기여가 없는데
  delta가 있는 경우.)
- **R4 (능력 주장 허용)**: R1–R3 전부 비발동일 때만. 프레이밍은 항상
  "큐레이션된 케이스 집합 위 능력 시연" — Bao/RUSBoost 등과의 벤치마크 비교
  가능 정확도/AUC 주장 금지 (N·과제 정의·기저율 상이).
- 복수 발동 시 우선순위: R1 > R3 > R2 (더 약한 주장이 이김).

## 5. 기준선 (동일 30사, PIT — §8-6 허용 결정론 공식)

- **Beneish M-Score** (8비율: DSRI GMI AQI SGI DEPI SGAI LVGI TATA), 임계
  −1.78과 −2.22 둘 다 보고. 연속 2개 연차 필요 — 컷오프 이전 최신 연차쌍.
- **Dechow et al. (2011) F-Score** (모델 1), F > 1 임계.
- 계산 불능 입력: 회사별로 어떤 비율이 왜 불능인지 기록 — **침묵 대체 금지**.
  결측 비율이 있는 회사는 해당 기준선 순위 비교에서 제외하고 N 명기.
- 각 기준선도 동일 순열 분리 검정 통과 여부 보고 ("M-Score 단독이 이 표본에서
  얼마 하는가").
