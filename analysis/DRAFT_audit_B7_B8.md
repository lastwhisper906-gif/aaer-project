## B7 — wave-1 "암기 우세(R3)" 5/8 헤드라인의 견고성

**문제**: 결론 R3는 8케이스 중 **정확히 5개**가 산입돼야 발동(≥5). SCOR는 임계 대비
+0.8pp(≈0.55 SEM)에 불과 → 재추첨 잡음으로 4/8이 되면 결론이 **R4(잔여 능력)로 반전**.

**방법**: 동결 k=5 재추첨(rp07_stats.per_case)의 orig/pert draw를 케이스별 복원추출해
delta_mean을 다시 만들고, 산입 규칙(stats.py와 동일)으로 8케이스 산입 수를 20,000회
부트스트랩. (denom = 본실행 orig_score − 대조군 중위 33; denom≤0이면 자동 산입.)

**결과**:

| 지표 | 값 |
|---|---|
| 관측 산입 (점추정) | **5 / 8** (발동) |
| **P(count ≥ 5)** | **0.60** |
| count 분포 | 3:0.10 · 4:0.29 · **5:0.39** · 6:0.19 · 7:0.02 |

케이스별 산입 빈도: MON 1.00(자동) · HTZ 0.96 · MRVL 0.76 · OFIX 0.73 · **SCOR 0.71** ·
LOGI 0.44 · ICON 0.11 · KHC 0.00.

**판독**: R3 발동(≥5)은 재추첨의 **약 60%**에서만 성립한다 — 즉 **≈40%의 재추첨에서는
4/8 이하로 떨어져 R4가 된다.** 헤드라인 "암기가 점수의 상당부를 설명"은 draw 잡음에
견고하지 않다. 경계를 흔드는 건 주로 SCOR(0.71)와 LOGI(0.44, 반대쪽).

**제안 문구(초안)**: *"wave-1에서 8개 중 5개가 암기-산입으로 분류돼 R3가 발동한다. 단
동결 k=5 재추첨 부트스트랩에서 이 5/8 발동은 P≈0.60으로, 결론이 R3와 R4 사이 경계에
있음을 명시한다(SCOR가 임계값 근처)."* — README/Issue #0 §7.6 각주 후보.

## B8 — 암기 dose-response가 대조군 주도 (fraud-side는 거의 불변)

**문제**: 헤드라인 dose-response는 "name-ID 50% → 21.9% → 0%로 떨어져도 분리 유지"이나,
**분리를 만드는 건 fraud-side**인데 하락은 대부분 대조군 쪽일 수 있다.

**결과** (동결 name_match 규칙):

| | fraud-side (분리 생성) | control-side |
|---|---|---|
| wave-1 | **3/8 = 37.5%** | 12/22 = 54.5% |
| wave-2 | **3/9 = 33.3%** | 4/23 = 17.4% |
| 이동 | **37.5% → 33.3% (거의 불변)** | 54.5% → 17.4% (급락) |

**판독**: 전체 name-ID 반감(50%→21.9%)은 **대조군 유명도 하락이 주도**한다. 정작
"탐지가 암기인가"에 걸리는 **fraud-side 인식은 wave1→wave2에서 사실상 그대로**(≈4pp).
따라서 wave1→wave2 구간은 "분리가 암기로 설명되지 않는다"는 **독립 근거가 약하고**,
암기-제거 증거의 실질은 **홀드아웃(fraud-side 0%, N=3)** 한 점에 가깝다. dose-response는
사실상 2점(암기가능 ≈35% vs 홀드아웃 0%)으로 읽는 게 정직하다.

**제안 문구(초안)**: *"name-ID를 fraud/control로 분리하면 분리를 만드는 fraud-side 인식은
wave-1 37.5% → wave-2 33.3%로 거의 불변이며, 전체 하락은 대조군(54.5%→17.4%)이 주도한다.
따라서 암기-제거의 핵심 증거는 홀드아웃(fraud-side 0%, N=3)이다."* — synthesis 캡션·
Issue #1 dose-response 각주 후보.
