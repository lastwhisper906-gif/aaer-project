# ANALYSIS_PLAN_WAVE2 — 사전 커밋 분석 계획 + 결론 규칙 (Wave 2 확장)

> Authored by Claude Code, pending human audit (GA-001 (b)). 2026-07-07.
> **지위: 이 파일은 wave-2의 어떤 채점 호출보다 먼저 커밋된다.** wave-2 점수가
> 존재하지 않는 시점의 커밋이라는 사실이 아래 결론 규칙의 가치다. 실행 코드는
> 동결 파이프라인 그대로 (`analysis/baselines.py` + `analysis/stats.py`,
> 결정론·시드 고정). 본 결과는 Claude 기반 단일 파이프라인에 한정된다 (§5-5).
> **동결 wave-1(8v22) 결과는 이 계획으로 재채점·재풀링되지 않는다** (RESULT
> IMMUTABILITY). wave-2는 독립 표본이다.

## 0. 이 계획의 원칙: 동결 계획과 **동일**

`analysis/ANALYSIS_PLAN.md`(wave-1)의 검정·효과크기·FPR 규칙·결론 규칙
R1–R4를 **변경 없이** wave-2 STANDALONE 표본에 적용한다. 새 검정을 발명하지
않는다. 아래는 그 재적용의 명세와, wave-2에만 존재하는 두 항목(pooled 2차,
암기 계층화)의 사전 고정이다.

## 1. 입력 정의 (사전 고정 — 동결 계획 §0과 동일)

- **LLM 점수** = 피평가자 `misstatement_probability` (0–100), 피평가자
  **claude-sonnet-5 핀 고정** (호출별 서빙 모델 검증, RP-10과 동일 격리).
- **1차 비교 프레임 = 원본 vs 원본**: wave-2 실험군 원본 점수 vs wave-2 대조군
  원본 점수. 근거: §5-2 동일 프로토콜.
- **2차(감도) 프레임 = 교란 실험군 vs 원본 대조군** (J14 비대칭 — 정체 암기
  기여 차단 방향). 두 프레임 병기, 결론 규칙 판정이 갈리면 **보수적(더 약한
  주장)이 헤드라인**.
- INCOMPLETE 케이스(재시도 후 실패): 1차 분석 제외 + N 명기 + 제외 민감도(최악
  대체: 대조군 100 / 실험군 0) 병기. **scoring order는 알파벳 사전 고정(§7)이라
  시간 절단 INCOMPLETE는 결과 독립적**이다.

## 2. 1차 통계 검정 (사전 고정 — 동결 계획 §1과 동일)

- **순열 검정, 평균 점수 차** (wave-2 fraud vs wave-2 control): 100,000 순열,
  seed = 20260707, 단측 (fraud > control), α = 0.05. **1차·primary.**
- **2차: Fisher exact** (단측), 2×2 = 그룹 × 플래그. **플래그 임계 = p ≥ 50**
  (동결 루브릭 임계 재사용 — 새 임계 발명 아님).
- **효과크기**: Cliff's delta (= rank-biserial 명기). **AUC** + 부트스트랩 95%
  CI (10,000 재표집, seed 20260707) — **N 소규모 불안정 라벨 의무**, 점 플롯이
  1차 시각자료.

## 3. FPR 보고 규칙 (사전 고정 — 동결 계획 §3과 동일)

**"0% FPR" 표기 금지.** FP=0/N_ctrl이면 단측 95% 상한 = 3/N_ctrl (rule of
three)로 보고. FP≥1이면 Clopper-Pearson 정확 구간. 관측 FPR은 항상 CI와 병기.

## 4. 결론 규칙 (R1–R4 — 동결 계획 §4와 **문자 그대로 동일**, wave-2 STANDALONE에 적용)

- **R1 (귀무)**: 1차 순열 p ≥ 0.05 → 헤드라인 = 귀무, 능력 주장 없이 발행.
- **R2 (기계 신호 복제)**: 어느 기준선 b∈{Beneish M, Dechow F}에 대해
  Spearman ρ(LLM,b) ≥ 0.7 **이고** 잔차 증분 검정 p ≥ 0.05 → "LLM이 기계 신호
  복제". (잔차 = wave-2 전체에서 LLM을 b에 OLS 회귀한 잔차, 그룹 평균차 순열
  100k, seed 동일.)
- **R3 (암기 지배)**: wave-2 실험군 중 **과반**에서 |Δmean(원본−교란 draw)| ≥
  0.5 × (해당 케이스 원본 점수 − wave-2 대조군 원본 중위값) → "점수 상당 부분이
  암기 반영". 분모 ≤ 0 케이스는 자동 산입.
  - **wave-2 교란 draw는 케이스당 1회**(§6 perturbation 규칙) — wave-1의 k=5
    평균과 달리 단일 draw다. 이 차이를 §5-2 감도로 명기하고, Δ의 잡음 대역은
    wave-1 케이스별 σ(RP-06)를 참조 밴드로 병기한다(하향 신뢰).
- **R4 (능력 주장 허용)**: R1–R3 전부 비발동일 때만. 프레이밍 항상 "큐레이션
  케이스 집합 위 능력 시연", 벤치마크 비교 정확도/AUC 주장 금지 (R4 프레이밍).
- 복수 발동 우선순위: **R1 > R3 > R2** (더 약한 주장이 이김). 발동 규칙은
  standalone·pooled 각각 기록.

## 5. 기준선 (동일 wave-2 firms, PIT — §8-6 허용 결정론 공식, 동결 계획 §5와 동일)

- **Beneish M-Score** (8비율), 임계 −1.78·−2.22 둘 다. 연속 2연차, 컷오프 이전
  최신 연차쌍. **Dechow (2011) F-Score** (모델 1), F > 1.
- 계산 불능 입력: 회사별 불능 비율·사유 기록 — **침묵 대체 금지**. 결측 회사는
  해당 기준선 순위 비교 제외 + N 명기.
- 각 기준선도 동일 순열 분리 검정 통과 여부 보고.

## 6. Perturbation 규칙 (사전 커밋 — 단일 규칙)

wave-2 **모든 실험군(fraud) 케이스에 정체-교란 draw 1회** (≈ 실험군 수만큼의
추가 피평가자 호출). 교란 = 동결 `scoring/perturbed_cases.json` 생성 규칙과
동일(정체 마스킹 + 케이스 고정 계수 금액 재스케일). 대조군은 교란하지 않는다
(원본만) — 동결 J14 비대칭 유지. **시간 열화 사다리 1순위 희생 항목**: 예산
초과 시 교란 draw를 가장 먼저 드롭하고 그 사실을 degradation 로그에 기록한다
(드롭 시 2차 프레임·R3 wave-2 판정은 "미측정"으로 명기, wave-1 R3에 의존
금지).

## 7. 고정 채점 순서 (사전 커밋)

**알파벳 티커 오름차순.** 실험군·대조군 각각 이 순서로 발사한다. 이 순서가
사전 커밋이므로 어떤 시간 절단(INCOMPLETE)도 결과 독립적(성능과 무관하게
알파벳이 결정)이다. 순서 = 실험군 생존 후보 알파벳 → 그 각각의 대조군 알파벳.

## 8. Pooled 2차 분석 (사전 고정 — 병기 전용)

pooled wave1+wave2 (실험군 8+wave2, 대조군 22+wave2)는 **2차로만** 보고하며
**항상 standalone과 나란히** 제시하고 **단독 헤드라인으로 쓰지 않는다**(DO NOT).
pooled에도 §2 검정·§4 결론 규칙을 동일 적용하되, wave-1 동결 점수는 **재계산
없이 커밋본 그대로 재사용**(재채점 아님) — pooling은 산술 결합일 뿐 RESULT
IMMUTABILITY 무침해. pooled 결과는 wave-1 대비 표본 확대의 방향(강화/희석)만
읽고, 발동 규칙은 standalone이 헤드라인을 결정한다.

## 9. 암기 계층화 비교 (wave-2 신규 — 사전 고정)

- **memorization_risk 태그(사전, 기계적)**: 동결 China-RTO 클러스터(T01/T03/
  T05/T06/T09/T30) 소속으로 게이트 생존한 케이스 = `memorization_risk=true`.
  그 외 실험군의 암기는 **사후 측정**(정체-교란 Δ + 인지 프로브 + 이름 예측
  프로브 — wave-1과 동일 계기).
- **계층 비교**: (i) memorization_risk=true vs false 실험군의 원본−교란 Δ 분포,
  (ii) 이름 예측 식별률 대조군 vs 실험군(wave-1 50%/54.5% 대비), (iii) 인지-이동
  검사(인지/비인지 대조군 점수 중위차 vs 사전 등록 10pp 바). 계층은 **기술
  통계로 보고**(N 소규모, 검정 아님), R3 판정의 해석 보조.

## 10. 부속 분석 (동결 계획 §6과 동일 방법, 결과 후 실행)

조기성(폭로 전 분기, d2 방법 재사용), 유형 계층화(AAER 모집단 분포 대비, wave-2
~17 실험군 점 갱신), 보정(ECE 10-bin), 암기 분해 산점도(Δ vs 원본, 잡음 밴드 =
wave-1 σ 참조), 이름 예측 프로브(동결 name_match 규칙 재사용).
