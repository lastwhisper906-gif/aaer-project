## 3. 동치성 테스트 설계 (승인 후 첫 작업 — 아직 미실행)

1. 대상: pilot 케이스 2건 (본채점 비병합 — pilot/ 경로, 카나리 무관).
2. 프로토콜: 동일 페이로드로 [하네스 경로 k=5] vs [raw API k=5 (temperature 미핀,
   현행과 동일 조건)] — p 점수 분포(중위·범위) 병기 + 스키마 통과율 + 서빙 모델 핀.
3. 판독 (사전 고정): 이것은 **게이트가 아니라 측정**이다 — 두 경로의 차이는
   "하네스 주입 효과 + 표본 분산"의 합이며 분리 불능(각 5 draw). 차이의 크기를
   문서화하고, per-case σ(E3 실측 3.2pp)의 2배를 넘는 계통 편차가 보이면
   발행물의 L-2 문단에 실측 수치로 병기한다.
4. 추가 1회 (권장): raw API temperature=0 k=5 — L-3(비결정론)의 실측 상한 문서화.
5. 예상 미터링: 2×5×2 + 10 = 30 호출 (종량 과금 — 소유자 예산 승인 대상).
