## 3. 누락(omission) 장르 근맹점 테스트 (1-3, GE 대상)

**가설**: 4종 스크린은 전부 "적극 조작이 남기는 비율 발자국"(AR 팽창, 마진 이상,
발생액 괴리)을 보는 도구다 — 추정·공시 누락형 조작은 원리적으로 잡기 어렵다.

**GE 실측 (컷오프 2017-07-20 이전 데이터)**:
1. **M/F-Score: 계산 자체가 불능** — GE는 무분류 연결 B/S(산업+금융)로
   `AssetsCurrent`/`LiabilitiesCurrent`를 보고하지 않아 AQI·LVGI·rsst·soft_assets가
   정의 불능. 금융복합기업이라는 구조적 사각지대.
2. **계산 가능한 스크린은 무신호**: C-Score 1/6 (자산성장·DSO 등 전부 정상 방향),
   Sloan +0.02 (플래그 임계 0.10의 1/5).
3. **기제 층위의 맹점(구조적)**: GE의 조작 실질은 보험 준비금 추정(LTC claim cost)
   악화의 미공시와 Power 이익의 원가 추정 변경 기여 미공시(33-10899 Summary·¶15/¶36)다.
   준비금 추정 변경은 매출·AR·재고·마진 어디에도 발자국을 남기지 않고, 보험 부채의
   과소는 '있어야 했던 증가분의 부재'라 전기 대비 지수로 검출 불가. **데이터가
   완전했어도 4종 스크린의 변수 정의상 신호가 생길 수 없는 위치의 조작**이다.
4. 동일 장르 교차 확인: LOGI(M −4.01, F 0.37, C 1)·HTZ(M/F 불능, C 0) — 누락 장르
   3건 전부 무신호. 반면 active 장르에서는 OFIX(M·F 동시 플래그)·VRX(F 1.97)·
   KHC(C 5, M −1.83) 신호 존재.

**결론(사전 가설로 고정)**: 4종 스크린은 omission-estimate 장르에 근맹점(near-blind).
Phase 3 분석 ③의 질문은 "LLM이 이 사각지대(각주·텍스트 채널)를 커버하는가"이며,
이 문서가 그 비교의 사전 기준선이다.

## 4. 한계 (정직 기록)

- **표본·생존 편향**: 본 표본은 해소 완료 집행 사건에서 의도적으로 추출 — 스크린
  성능의 모집단 일반화 불가. (전 보고서 공통 의무 문단.)
- 태그 폴백의 한계: `lt_debt` 등 일부 개념은 발행사 태깅 관행에 따라 미포착 가능 —
  미발견 시 0 폴백+기록이며, 이는 LVGI/rsst를 보수(무신호) 방향으로 치우치게 한다.
- XBRL 초기 연도(2009-2011 컷오프 케이스)는 연차 2개 확보가 한계 — t−2 필요 변수
  (F-score 일부)는 근사+노트로 처리 (T07).
- 단일 실행·결정론이므로 분산 없음. 단, **입력**(companyfacts 아카이브)은 SEC 측
  갱신에 따라 달라질 수 있어 수집 시각의 스냅샷임을 명기 (매니페스트로 고정).
